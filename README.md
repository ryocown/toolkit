# Personal Toolkit

A collection of strategic finance and compliance tools.

## Tools Included

- **Tax Roadmap**: Strategic compliance and asset defense dashboard for Japan residents.
- **Japan Income Tax Calculator**: Estimate take-home pay including RSU and stock options.

## Development

```bash
npm install
npm run dev
```

## Deployment

This project is configured for GitHub Pages. To deploy:

1. Build the project: `npm run build`
2. The output will be in the `dist` folder.
3. Host the contents of `dist` on GitHub Pages.
